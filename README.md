# emo-dules

このリポジトリは、様々なプロジェクトで再利用可能なラッパークラスやユーティリティモジュールを管理するためのコレクションです。
標準APIの複雑さを隠蔽し、より使いやすいインターフェースを提供することを目的としています。

## 概要

`emo-dules` は、開発プロセスにおいて頻繁に必要となる機能や、ボイラープレートコードを削減するためのラッパーを集約しています。
各モジュールは独立して利用できるように設計される予定です。

## 収録モジュール (予定・開発中)

### 🌐 HTTP Client
JavaScript の標準 `fetch` API をラップしたHTTPクライアントです。
- リクエスト/レスポンスのハンドリングの簡略化
- エラーハンドリングの標準化
- インターセプターやタイムアウトなどの拡張機能（予定）

*(今後、データベースラッパーやロギングユーティリティなども追加予定)*
